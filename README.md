# Rest API .NET core Web API task

## How to run

### Visual Studio 
1. clone repo
2. click on api2.sln

### Command  line
1. clone repo
2. navigate to folder containing api2.sln
3. Run
  dotnet restore # Download NuGet packages
  dotnet run     # Build and run project

### VS Code
1. Clone repo
2. Open folder in VS Code
3. Run
   dotnet run
